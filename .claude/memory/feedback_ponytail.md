Sempre invocar o skill `ponytail:ponytail` antes de escrever qualquer código.

**Why:** Usuário quer garantir o mínimo de código necessário, sem over-engineering, em todos os projetos.

**How to apply:** Em qualquer tarefa que envolva criar ou modificar código, ativar Ponytail (intensidade `full` por padrão) antes de implementar. Só pular se o usuário pedir explicitamente para ignorar o Ponytail.
